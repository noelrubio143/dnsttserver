**One-command installation:**
```
curl -sSL https://raw.githubusercontent.com/noelrubio143/dnsttserver/refs/heads/main/dnstt -o /tmp/stunnel-menu.sh && sudo mv /tmp/stunnel-menu.sh /usr/local/bin/dnstt && sudo chmod +x /usr/local/bin/dnstt && echo "Installation complete. Type 'dnstt' to open the SSH + Stunnel menu."
```
# UNINSTALL SLIPSTREAM
```
bash <(curl -Ls https://raw.githubusercontent.com/noelrubio143/dnsttserver/refs/heads/main/dnstt) uninstall
```
