**One-command installation:**
```
curl -sSL https://raw.githubusercontent.com/noelrubio143/dnsttserver/refs/heads/main/dnstt -o /tmp/dnstt && sudo mv /tmp/dnstt /usr/local/bin/dnsttserver && sudo chmod +x /usr/local/bin/dnsttserver && echo "Installation complete. Type 'dnstt' to open the SSH + dnstt menu."
```
# UNINSTALL SLIPSTREAM
```
bash <(curl -Ls https://raw.githubusercontent.com/noelrubio143/dnsttserver/refs/heads/main/dnstt) uninstall
```
